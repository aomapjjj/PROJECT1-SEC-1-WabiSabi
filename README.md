## Application Features

1. **Feature 1: เลือกโหมดในการเล่นเกมบิงโกได้ 2 โหมด**
   - Mode Line Pattern (บิงได้แบบเส้นแนวตั้ง, แนวนอน, แนวทแยง)
   - Mode Blackout Pattern (บิงแบบทั้งแผ่น)
2. **Feature 2: สุ่มเลขอัตโนมัติ**
   - เลขที่ออกมาเป็นแบสุ่มเลขอัติโนมัติ เพื่อไม่ให้ผู้เล่นใช้เวลาไปกับการกดมากเกินไป
3. **Feature 3: หยุด-ไปต่อในการสุ่มเลข ระหว่างเล่นเกม**
  - ให้ผู้เล่นสามารถที่จะหยุดเกมเพื่อดูเลขย้อนหลังได้ระหว่างที่กำลังเล่นเกม แต่ไม่สามารถกดใบบิงโกในขณะหยุดเกมได้
4. **Feature 4: สามารถดูเลขที่ถูกสุ่มไปแล้วย้อนหลังจากเลขทั้งหมด**
  - นอกจากจะหยุดเกมเพื่อดูได้แล้ว ยังสามารถดูเลขทั้งหมดที่ถูกสุ่มไปจากตารางเลขทั้งหมด
5. **Feature 5: สามารถสับเปลี่ยนเลขบนใบบิงโก**
   - เมื่อเข้าสู่เกมก่อนที่จะเริ่มเกมแล้ว ผู้เล่นสามารถสับเปลี่ยนเลขใบบิงโกทั้งหมดเพื่อเลือกเลขที่ถูกใจได้

## User Guide
#### 1. เมื่อผู้เล่นเข้ามาที่หน้าเกมแล้ว ผู้เล่นสามารถเลือกโหมดที่จะเล่นได้ 2 โหมด คือ

- **Line:** ผู้เล่นจะบิงโกได้ก็ต่อเมื่อ **คลิกเลขที่ตรงกัน 5 ตัวบนใบบิงโกในรูปแบบได้แบบหนึ่ง**

- **Blackout:** ผู้เล่นจะบิงโกได้ก็ต่อเมื่อ **กดทุกเลขบนใบบิงโก**
<img width="1440" alt="Screenshot 2567-09-10 at 12 57 39" src="https://github.com/user-attachments/assets/6232a555-1006-40b5-9816-f271410c155a">


#### 2. เมื่อเลือกโหมดที่ต้องการเล่นได้แล้ว ผู้เล่นสามารถ shuffle เพื่อเปลี่ยนเลขบนใบบิงโกได้ เมื่อผู้เล่นพร้อมแล้วให้กดปุ่มเล่นเกมที่อยู่ตรงกลาง
